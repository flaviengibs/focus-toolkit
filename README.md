# Focus Toolkit – Extension Chrome

Une extension Chrome simple de productivité regroupant :
- une todo list locale
- des notes rapides
- la sauvegarde d’onglets

---

## 🚀 Fonctionnalités

### ✔️ To-do list
Ajoute et supprime des tâches rapidement. Les données sont stockées localement.

### ✔️ Notes rapides
Permet de stocker des idées ou textes courts directement dans le navigateur.

### ✔️ Sauvegarde d’onglets
Sauvegarde l’onglet actif pour le retrouver plus tard.

### ✔️ Stockage local
Toutes les données sont enregistrées via `chrome.storage.local` (hors ligne).
